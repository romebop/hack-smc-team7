Getting Started
---------------
Git:

* git clone https://github.com/romebop/hack-smc-team7.git
* cd hack-smc-team7

Set up environment:

* pip install virtualenv
* virtualenv env
* . env/bin/activate
* pip install -r requirements.txt

App:

* python server.py
