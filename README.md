## Getting Started ##
Setup Git:

* git clone https://github.com/romebop/hack-smc-team7.git
* cd hack-smc-team7

Get Flask:

* [install flask (& virtualenv)](http://flask.pocoo.org/docs/0.10/installation/)
* ([familiarize flask](http://flask.pocoo.org/docs/0.10/quickstart/))

Run App Locally:

* python wi-finder.py
