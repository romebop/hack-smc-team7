## Getting Started ##
* git clone https://github.com/romebop/hack-smc-team7.git
* cd hack-smc-team7
* Start Web Server
 * On python2.x: python -m SimpleHTTPServer
 * On python3.x: python -m http.server
* Go to: http://localhost:8000
